# LinkedIn Job Market Analytics
An Advanced MySQL project focused on analyzing hiring trends, salary insights, skill demand, and workforce patterns using relational databases and analytical SQL queries.

## Features
- Relational Database Design
- Advanced JOIN Queries
- Window Functions
- CTEs (Common Table Expressions)
- Subqueries & CASE Statements
- Salary & Hiring Trend Analysis
